# Media Kit

## Product name

DevMaid

## Short description

Free preview-first storage cleanup for developer Macs.

## Full description

DevMaid is a native macOS cleanup tool built around the storage problems developer machines hit most often: Xcode build junk, Docker data, `node_modules`, `.venv`, editor caches, package caches, browser-test binaries, and project artifacts. It shows exactly what it finds before cleanup, quarantines instead of hard deleting, and supports undo from history.

## Key feature bullets

- native macOS app plus `devmaid` CLI
- preview-first, quarantine-first, undo-capable cleanup
- risk labels for cleanup categories
- exclusions list for projects or paths you want skipped
- weekly reclaimable-space trend from local scan history
- low-space and reclaimable-spike alerts
- Thai and English UI
