## Short launch post

DevMaid is a free Mac cleanup app built for developers.

It scans Xcode junk, Docker data, `node_modules`, `.venv`, editor caches, package caches, and build artifacts with a preview-first workflow that shows exactly what you are about to clean. Cleanup is quarantine-first and undo-capable, and there is a matching CLI for automation too.

## Longer launch post

I built DevMaid because generic Mac cleaners usually miss the storage problems that actually hurt developer machines.

This app focuses on Xcode, Docker, `node_modules`, `.venv`, editor caches, package caches, project artifacts, browser-test binaries, and other rebuildable dev junk. The goal was not to make another one-click cleaner. The goal was to make cleanup visible and safe.

DevMaid shows category, size, path, and risk labels before you clean anything. Cleanup goes through quarantine first, undo is built in, and there is both a native macOS app and a CLI.

Current release highlights:

- free desktop app and free CLI
- preview-first, quarantine-first cleanup
- Thai and English UI
- exclusions list
- local smart alerts and weekly trends
- startup-item visibility and launch-at-login control for DevMaid itself
