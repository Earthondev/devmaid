# Support

Thanks for using DevMaid.

Use the right channel so people can get help quickly and maintainers can keep the project healthy.

## Where to ask for what

- Bug reports:
  Open a GitHub Issue with reproduction steps, macOS version, DevMaid version, and the command you ran.
- Feature requests:
  Open a GitHub Issue or Discussion with the cleanup target you want added and why it matters for developer workflows.
- Usage questions:
  Prefer GitHub Discussions if enabled. If not, open an Issue and label it as a question.
- Security issues:
  Do not post publicly first. Follow [SECURITY.md](/Users/earthondev/Desktop/Devmaid/SECURITY.md).

## Before opening an issue

- Run `devmaid help`
- Check [README.md](/Users/earthondev/Desktop/Devmaid/README.md)
- Check [DISTRIBUTION.md](/Users/earthondev/Desktop/Devmaid/docs/DISTRIBUTION.md) if the problem is about packaging or installation
- Include whether Terminal has Full Disk Access if the issue involves missing scan results

## What helps maintainers most

- exact command used
- full error text
- whether the problem is reproducible
- whether the issue affects only one category or all scans
- sample paths if the problem involves a missed cache or dependency folder
